# YuAiCodeMother Open Source Project

## Project Introduction

`YuAiCodeMother` is a Java-based open source project aiming to achieve automated code generation through AI technology. This project combines AI models with modules for code parsing, saving, and building, capable of generating various types of code files such as HTML, CSS, JavaScript, etc., based on natural language descriptions provided by users, and supports building Vue projects.

The project is primarily aimed at developers and AI engineers, providing a complete set of code generation, saving, deployment, and management functions, suitable for scenarios such as rapid prototyping and low-code platform construction.

---

## Key Features

- **AI Code Generation**: Generate HTML, CSS, JS, or Vue project code based on natural language input using AI models.
- **Code Parsing and Saving**: Support parsing generated code into structured objects and saving them as files.
- **Vue Project Building**: Provide the capability to build Vue projects asynchronously.
- **Streaming Processing and Response**: Support streaming responses, returning generated code snippets in real-time.
- **Tool Integration**: Integrate tools for file reading, writing, deletion, and modification, supporting interaction with the AI generation process.
- **User and Permission Management**: Support user registration, login, and permission control features.
- **Chat History Records**: Record interaction history between users and AI.
- **Rate Limiting Mechanism**: Prevent request overload, supporting Redis-based rate limiting strategies.

---

## Technology Stack

- **Backend**: Java + Spring Boot
- **AI Model**: LangChain4j + OpenAI Streaming Chat Model
- **Database**: MySQL (operated via MyBatis Plus)
- **Cache**: Redis (used for chat memory and rate limiting)
- **File Storage**: MinIO (used for screenshots and file storage)
- **Frontend**: Vue.js (optional, used for generating frontend projects)
- **Build Tool**: Maven

---

## Project Structure Overview

- **AI Generation Module**: Responsible for interacting with the AI model to generate code.
- **Code Parsing Module**: Parses code generated by AI, extracting HTML, CSS, JS, and other content.
- **Code Saving Module**: Saves parsed code into files.
- **Vue Build Module**: Supports building Vue projects.
- **Tool Module**: Provides tools for file reading, writing, deletion, and modification.
- **User System**: Supports user registration, login, and permission control.
- **Rate Limiting Module**: Prevents request overload.
- **Chat History Module**: Records interactions between users and AI.
- **Deployment Module**: Supports project deployment and static resource access.

## Quick Start

### Environment Requirements

- Java 17+
- Maven 3.8+
- MySQL 8.0+
- Redis 6.0+
- MinIO (optional)
- Node.js (used for building Vue projects)

### Installation Steps

1. **Clone the Project**

   ```bash
   git clone https://gitee.com/cabbage12/codeless.git
   cd codeless
   ```

2. **Configure Database**

   Modify the database connection information in `application.yml`:

   ```yaml
   spring:
     datasource:
       url: jdbc:mysql://localhost:3306/yuaicodemother?useSSL=false&serverTimezone=UTC
       username: root
       password: your_password
   ```

3. **Initialize Database**

   Execute the SQL script to create table structures (the script is located in `src/main/resources/sql`).

4. **Configure AI Model**

   Configure the OpenAI API Key and model parameters in `application.yml`:

   ```yaml
   langchain4j:
     open-ai:
       streaming-chat-model:
         api-key: your_openai_api_key
         model-name: gpt-3.5-turbo
   ```

5. **Start the Project**

   ```bash
   mvn spring-boot:run
   ```

6. **Access the API**

   After starting, visit `http://localhost:8080/swagger-ui.html` to view the API documentation.

## Usage Examples

### Generate HTML Code

```bash
POST /app/chat/gen/code
Content-Type: application/json

{
  "appId": 1,
  "message": "Generate a simple calculator interface"
}
```

### Deploy Project

```bash
POST /app/deploy
Content-Type: application/json

{
  "appId": 1
}
```

### Retrieve Generated Code

```bash
GET /static/{deployKey}/**
```
